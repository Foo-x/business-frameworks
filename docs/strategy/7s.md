# 7S

経営に必要な7つの資源。  
短い時間で変更できる**ハードの3S**と、変更に時間がかかる**ソフトの3S**に分けられる。

## ハードの3S
- 戦略(Strategy)
  - 競争優位性
  - 選択と集中
- 構造(Structure)
  - 指揮系統
- システム(System)
  - 給与体系
  - 評価制度

## ソフトの3S
- 価値観(Shared Value)
  - 経営理念
- 能力(Skill)
  - 営業力
  - 技術力
- 人材(Staff)
  - 人事
  - リーダーシップ
  - 教育
- 文化(Style)
  - 社風
  - 意思決定フロー
